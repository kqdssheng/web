🛠️软件百宝箱

收藏 小巧、单文件、可直接运行的软件（仅限于windows）；

记录 需要安装/解压、但功能强大的应用（不限于windows，还包括安卓应用、浏览器扩展等）。

收藏​

类别说明
Cmd命令行工具集
Msic软件来源五花八门的工具集
NirSoft来自 NirSoft 官方提供的系统维护工具集
Sordum来自 Sordum 官方提供的系统维护工具集
SysinternalsSuite来自微软官方提供的系统维护工具集

记录​

  1. Windows 类
  2. MobaXterm：远程终端登录（功能齐全：内置多种服务器角色、本地端支持伪Linux命令）👍
  3. SecureCRTPortable：远程终端登录（命令多窗口同时发送比MobaXterm体验要好）
  4. Proxifier：分应用代理👍
  5. Scoop：Windows命令行包管理器（妙用无穷，包山包海）👍👍
  6. Choco：Windows命令行包管理器（官方仓库软件种类繁多，但命令使用不太友好：软件安装路径不固定，安装过程容易报错）
  7. Vfox：跨平台版本版本管理器（支持众多常见的SDK，如java、python、dotnet等）
  8. anaconda3：针对 python 的版本管理器
  9. BitComet：比特下载器（适合搭代理下载国外镜像，国内资源还是以迅雷为主）
  10. FileZilla FTP Client：FTP客户端
  11. Navicat：数据库图形界面客户端（支持多种数据库的连接）
  12. SyncTrayzor：跨设备文件同步工具（区别于常规的云端存储同步，工作方式类似于去中心化的P2P技术，最重要的是适用于局域网或互联网的网络环境。配合安卓客户端Syncthing可实现手机电脑文件随时同步的效果）👍
  13. RaiDrive：云存储本地挂载（类似于windows磁盘映射，但这个支持的存储类型丰富）
  14. 科来网络分析系统：相当于wireshark抓包工具的补充，wireshark只能读、分析数据包，而这款软件却支持分析、编辑数据包。
  15. 坚果云：跨设备文件同步共享（支持WebDAV，算是国内外云盘中支持WebDAV最好的）👍
  16. 印象笔记：云笔记（支持网页剪藏）
  17. Vnt：异地组网、内网穿透工具（配合chisel可更高效的实现ip转发、端口映射功能）👍
  18. Chisel：基于TCP/UDP的流量通道👍
  19. ShareX【FastStone Capture】：截图录屏
  20. Bandicam：体验非常好的一款录屏软件
  21. Listary：文件快速搜索（个人觉得比everything好用）
  22. KeePass：密码管理器
  23. Treesize Free【WizTree】：磁盘空间分析
  24. FreeFileSync：文件夹对比及同步（支持对本地文件进行监视，进而实时同步，即RealTimeSync功能）👍
  25. Kleopatra：文件加解密（相当于Linux下的GPG）
  26. HandBrake：视频格式转换器（容量高压缩比，清晰度低损耗）👍
  27. ShanaeEcoder：视频压制软件（又称FFmpeg最强套壳）
  28. Autoit v3：图形界面脚本语言（仅适用Windows系统）
  29. Pycharm：Python IDE工具（也适用于Shell/PowerShell编写，需安装扩展Shell Script/PowerShell）
  30. Mark Text：Markdown编辑器👍
  31. GitExtensions/TortoiseGit：Git图形界面（功能强劲，将命令行git的大部分功能都以图形化显示）👍
  32. Cygwin64：Windows下类UNIX模拟环境（也可以使用Windows中WLS：Ubuntu、kali）
  33. Sandboxie：软件沙盘（相比于打开虚拟机来测试来路不明的软件来说这个更轻便）
  34. VMWare Workstation Pro：虚拟机（感觉比Virtual Box好用）
  35. Wsusoffline：Windows系统离线升级（特殊情况下可能会用到）
  36. 格式工厂：功能全面的格式转换器（支持转换几乎所有主流的多媒体文件格式）
  37. QTTabBar：资源管理器强化扩展
  38. VLC：媒体播放器（支持功能插件扩展）
  39. Ditto：剪切板强化扩展
  40. Ventoy：U盘启动制作工具（配合优启通、微PE等PE ISO镜像，可以实现通过一个U盘进入多个PE环境的效果，进而随意选择自己喜爱的PE环境进行系统的安装）👍
  41. Shimeji：PC桌面宠物
  42. PortableApps.com：可携式绿色软件平台（可跟随U盘随查随用的各种绿色软件整合平台）
  43. RegWorkshop：功能更强的注册表管理器
  44. Lris Pro：防蓝光护眼软件
  45. Process Hacker：功能更强的任务管理器👍
  46. JavE：全能型Ascii绘图工具（用字母画图写字）👍
  47. ABBYY FineReader PDF：相当于一个离线版的OCR文字识别工具
  48. 方方格子、iSlide、小恐龙公文排版助手：Office办公三件套强化插件
  49. QuickLook：快速预览所支持文件的内容（基本上常见的文件均支持）
  50. smartpackager：将需要多次点击安装的exe安装包直接转换为点击一次即安装的msi安装格式
  51. Portable Python-3.10.0：便携版python IDE编写调试环境
  52. AcronisTrueImage：ATI系统备份还原（类似Windows自带系统还原点创建工具）
